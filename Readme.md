# Brazilian Dynasties Maps

Map pack for the [Brazilian Dynasties](https://liquipedia.net/ageofempires/Brazilian_Dynasty) tournament, hosted by [Onimaru Esports](https://teamonimaru.com/).
Install the mod by searching for "Brazilian Dynasties Maps" in the in-game mod center, or by using the following link:

[https://www.ageofempires.com/mods/details/469602/](https://www.ageofempires.com/mods/details/469602/)

The map pool includes the following 11 maps:

- Amazônia ([T-West](https://liquipedia.net/ageofempires/T-West))
- Araucárias ([T-West](https://liquipedia.net/ageofempires/T-West))
- Arena Costeira ([TheMadCADer](https://liquipedia.net/ageofempires/TheMadCADer))
- Fernando de Noronha ([T-West](https://liquipedia.net/ageofempires/T-West))
- Festa da Pólvora ([T-West](https://liquipedia.net/ageofempires/T-West))
- Lençóis Maranhenses ([T-West](https://liquipedia.net/ageofempires/T-West))
- Monte Pascoal ([T-West](https://liquipedia.net/ageofempires/T-West))
- Pantanal ([T-West](https://liquipedia.net/ageofempires/T-West))
- Serra dos Carajás ([TheMadCADer](https://liquipedia.net/ageofempires/TheMadCADer))
- Sertão ([T-West](https://liquipedia.net/ageofempires/T-West))
- Vale da Selva ([T-West](https://liquipedia.net/ageofempires/T-West))

The maps Arena Costeira and Serra dos Carajás were originally named [Coast Arena](https://liquipedia.net/ageofempires/Coastarena) and [Terrace](https://liquipedia.net/ageofempires/Terrace), respectively.
These maps are based on the original scripts by TheMadCADer, check out his YouTube channel for videos about map scripting: [https://www.youtube.com/@MikeHasselbach](https://www.youtube.com/@MikeHasselbach)

Lençóis Maranhenses is an original map and should not be confused with the [3v3 map of the same name](https://liquipedia.net/ageofempires/Len%C3%A7%C3%B3is_Maranhenses) by [Huehuecoyotl22](https://liquipedia.net/ageofempires/Huehuecoyotl22) used for the [World Desert Championship 2](https://liquipedia.net/ageofempires/World_Desert_Championship/2).
